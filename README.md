# Trip-Expenditure-System

This is a simple Trip Expenditure System which can help you split the with your friends during the trips.

# Features
  - Trip-based logging
  - Password protection
  - Multi-currency support

### Installation Requirements
* MySQL 5.5+
* PHP 5.6+
